개인정보처리방침 — JLPT10M

시행일: 2026-06-23 · 본 문서는 법률 자문이 아닌 일반 안내문입니다.

JLPT10M(이하 "본 앱")은 하루 10분 일본어 학습 서비스입니다. 본 앱은 이용에 필요한 최소한의 정보만 수집하며, 아래와 같이 처리합니다.

1. 수집하는 정보

  • 이메일 주소 — Firebase Authentication 로그인/계정 관리용.
  • Firebase UID — 사용자 식별용(이메일이 아닌 무작위 식별자).
  • 사용자 활동 요약(userActivity) — firstSeenAt, createdAt, lastSeenAt, lastEventType, signedIn, sessionCount, totalActiveMs, lastRoute, platform, appVersion. (가입/접속 시각, 세션 수, 대략적 이용시간 등 요약 정보)
  • 피드백(feedback) — 사용자가 직접 입력한 의견: rating, good, bad, wish, bug, contactOk, createdAt, platform, appVersion, uid.

2. 수집하지 않는 정보

  • 비밀번호 원문 — 저장하지 않습니다(Firebase 가 인증을 관리).
  • 학습 답변 원문 — 정답/오답 여부 외 입력 원문을 저장하지 않습니다.
  • 음성 인식(STT) 음성/인식 원문 — 저장하지 않습니다.
  • 결제 정보, 위치 정보, 연락처, 사진, 파일.

상세 행동 로그(개별 클릭/이벤트 기록)는 수집하지 않으며, 활동은 위 "요약" 형태로만 저장됩니다.

3. 이용 목적

  • 로그인 및 계정 관리
  • 학습 앱 운영(서비스 제공)
  • 오류 및 사용성 개선
  • 사용자 피드백 확인
  • 부정 사용 방지

4. 제3자 제공 및 처리 위탁

본 앱은 수집한 정보를 제3자에게 직접 판매하거나 제공하지 않습니다. 다만 서비스 운영을 위해 Google Firebase(Authentication, Realtime Database) 인프라를 사용하며, 데이터는 해당 인프라에 저장·처리됩니다.

5. 보관 기간

  • 계정이 유지되는 기간 동안 보관합니다.
  • 사용자가 삭제를 요청하면 가능한 범위에서 삭제합니다.
  • 피드백은 운영/개선 목적에 따라 보관될 수 있습니다.

6. 사용자 권리 및 데이터 삭제 요청

사용자는 자신의 계정 및 데이터 삭제를 요청할 수 있습니다. 앱 내 설정 화면 → "계정 및 데이터 삭제 요청" 버튼(또는 아래 이메일)으로 요청하실 수 있습니다.

  • 요청에 포함되는 정보: 가입 이메일, 사용자 ID(uid).
  • 삭제 대상: 계정(Firebase Authentication), 활동 요약(userActivity), 피드백(feedback) — 가능한 범위에서 삭제.
  • 처리 방식: 운영자가 확인 후 수동 삭제(자동 삭제 아님). 비밀번호·학습 답변 원문·STT 원문은 애초에 저장하지 않습니다.

문의/요청: jlpt10m@gmail.com

7. 아동의 개인정보

본 앱은 일반 학습 앱이며 아동을 주 대상으로 하지 않습니다. 만 14세 미만 이용자는 보호자(법정대리인)의 동의 하에 이용해 주시기 바랍니다.

8. 보안

  • 데이터 접근은 Firebase Realtime Database 보안 규칙(Rules)으로 제한됩니다 — 사용자는 자신의 활동만 기록하며, 피드백/활동 전체 조회는 관리자만 가능합니다.
  • 관리자 접근은 Firebase UID 기준으로 제한됩니다.
  • 통신은 HTTPS 로 암호화됩니다.
