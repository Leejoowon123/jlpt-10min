JLPT N5부터 N2까지, 단어·문법·독해·청해·이야기·회화를 한 곳에서 학습하세요.

- 단어와 문법을 예문·이야기와 함께 익히고 싶은 분

■ 주요 기능
- 단어 학습: 이미지 카드와 단계형(노출→회상→확인→퀴즈) 학습, 후리가나 표시
- 문법: 핵심 문형과 헷갈리는 문법 비교

- 이야기·단편: 문장별/전체 재생, 로마자·해석 표시 옵션
- 회화: 주제별 문장 연습
- 발음 듣기: 기기 내장 음성으로 일본어 발음 재생
- 복습: 틀린 항목 다시 보기
- 다크/라이트 테마, 후리가나/로마자 토글

■ 콘텐츠
- N5~N2 단어·한자·문법·독해·청해·이야기가 앱에 포함되어 있어, 첫 실행 후에는 네트워크 없이도 학습할 수 있습니다.
- 학습 콘텐츠는 자체 제작한 오리지널 자료입니다.

■ 계정

- 학습 답변 원문, 음성 인식 원문, 비밀번호는 저장하지 않습니다.
- 자세한 내용은 개인정보처리방침을 확인해 주세요.

■ 안내(비공식)
- 본 앱은 JLPT 시험을 주관하는 기관, 일본 정부, 공식 시험 운영 주체와 아무런 관련이 없는 비공식 학습 앱입니다.

문의: jlpt10m@gmail.com
