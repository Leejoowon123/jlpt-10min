# 이미지 자산 라이선스 기록

모든 이미지 자산의 출처/라이선스를 이 파일에 기록한다.
**기록 없는 이미지는 사용 금지.** smoke 가 stories 의 `coverImage.licenseId` 와
이 문서의 asset id 일치를 검증한다.

## 정책

우선순위:
1. **직접 만든 SVG/일러스트** (가장 안전 — 현재 전부 이 방식)
2. Public Domain / CC0
3. 상업적 사용 가능 + attribution 조건이 명확한 CC BY
4. Unsplash/Pexels/Pixabay 자체 라이선스 — 개별 약관 확인 후 제한적으로

금지:
- 외부 이미지 hotlink (프로젝트에 저장한 파일만 사용)
- 인물 사진·상표/로고가 보이는 사진 (초상권/상표권)
- 기출/교재/상용 앱 이미지 복사

## 외부 무료 이미지 도입 시 체크리스트
- [ ] 라이선스 페이지 URL 을 source 에 기록했는가
- [ ] 상업적 사용 + 수정 허용인가
- [ ] attribution 요구 시 attribution text 를 기록했는가
- [ ] 파일을 assets/ 아래 저장했는가 (hotlink 금지)
- [ ] 인물/로고/상표 미포함인가
- [ ] downloaded date 를 기록했는가
