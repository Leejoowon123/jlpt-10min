# 앱 설치 안내 (PWA)

**JLPT10M** 은 **PWA(설치형 웹앱)** 입니다. 브라우저에서 홈 화면/바탕화면에 추가하면 일반 앱처럼 전체화면으로 실행되고, 재방문 시 더 빠르게 열립니다. (설치 시 앱 이름 **JLPT10M**, 먹+주홍 아이콘)

> 배포 URL: <https://leejoowon123.github.io/jlpt-10min/>

## 설치 방법

### Android — Chrome
1. Chrome 으로 배포 URL 접속.
2. 주소창 오른쪽 **⋮ 메뉴 → "앱 설치"** 또는 하단 배너의 **"홈 화면에 추가"** 탭.
3. 확인 → 홈 화면에 아이콘 생성. 아이콘을 누르면 전체화면 앱으로 실행.

### PC — Chrome / Edge
1. Chrome 또는 Edge 로 배포 URL 접속.
2. 주소창 오른쪽 **설치 아이콘(⊕ / 모니터 모양)** 클릭 → **설치**.
   - 안 보이면 **⋮ 메뉴 → "앱 설치"** / Edge 는 **앱 → 이 사이트를 앱으로 설치**.
3. 독립 창으로 실행되고, 작업표시줄/런처에 등록됩니다.

### iPhone / iPad — Safari
1. Safari 로 배포 URL 접속(Chrome iOS 는 설치 미지원 — Safari 사용).
2. 하단 **공유 버튼(□↑) → "홈 화면에 추가"**.
3. 이름 확인 → **추가**. 홈 화면 아이콘으로 실행.

## 로그인 / 온라인 안내

- **첫 사용은 온라인 로그인이 필요합니다.** 처음 실행하면 로그인/회원가입 화면이 나오며, 이메일 + 비밀번호(6자 이상)로 가입하면 바로 로그인됩니다.
- **로그인 세션이 유지되면 오프라인에서도** 앱 화면과 학습(단어·문법·독해·청해·이야기·회화)이 동작합니다. (단, 첫 로그인/로그아웃 후 재로그인은 온라인 필요)
- 학습 진행/복습 기록은 기기에 저장됩니다(localStorage). 기기를 바꾸면 이어지지 않습니다(현재 동기화 미지원).

## 기능 한계 안내

- **음성(TTS)** 과 **음성 인식(STT)** 은 브라우저/기기의 무료 Web Speech 기능에 의존합니다. 미지원 기기에서는 발음 재생이 안 되거나, 회화에서 텍스트 입력 폴백으로 동작합니다.
- **행동 로그**(학습 시작 등 최소 기록)는 온라인에서만 전송되며, 실패해도 학습은 멈추지 않습니다. 답변 원문·음성 원문·이메일·비밀번호는 저장하지 않습니다.
- 오프라인 중에는 로그인/회원가입이 불가하며, 로그인 화면에 "온라인 연결 필요" 안내가 표시됩니다.
